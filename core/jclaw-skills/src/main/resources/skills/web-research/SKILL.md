# Web Research

Guidelines for performing web research and synthesizing information:

## Searching
- Use **WebSearch** to find current information on a topic.
- Formulate specific, targeted search queries rather than broad ones.
- Try multiple query variations if initial results are insufficient.

## Fetching Content
- Use **WebFetch** to retrieve and analyze specific web pages.
- Extract the relevant information and discard boilerplate.
- Verify claims across multiple sources when possible.

## Synthesis
- Summarize findings concisely, leading with the most important information.
- Cite sources by URL so the user can verify.
- Distinguish between facts and opinions or speculation.
- Note when information may be outdated or conflicting across sources.
- Present structured comparisons when evaluating alternatives.
